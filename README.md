# 🧑‍⚕️현대인의 건강을 위한 플랫폼 MediChart
[프로젝트 소개] <br/>
-Medichart는 건강검진이 의무화가 됐음에도 불구하고 바쁜 일상으로 인해 자신의 건강을 놓치는 현대인들을 위해 기획된 질병 예측 및 건강진단서 해석 플랫폼입니다.
